# Support
## Getting Help
For questions and usage help:
1. Check documentation in `AndroidModApp/README.md` and `AndroidModApp/docs/`.
2. Search existing issues first.
3. Open a new issue using the appropriate template.
## What to Include
- What you are trying to do
- Commands executed
- Full error output
- Environment details (OS, JDK, Gradle, Android Studio)
## Maintenance Expectations
- Issues are triaged within 2 business days.
- Pull requests are acknowledged within 2 business days.
- Critical regressions are prioritized.
